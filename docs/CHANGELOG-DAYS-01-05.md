# Day-by-day baseline

## Day 1 — Enterprise AI foundations

- Multi-module Maven architecture and stable domain ports
- Provider-neutral `AiModelPort` with local Ollama adapter
- Health, metrics, and configuration foundations

## Day 2 — Retrieval-augmented generation

- Knowledge document, metadata, chunk, search, citation, and answer models
- Section-aware chunking and hybrid keyword/phrase retrieval
- Grounded prompt assembly and citation-bearing responses

## Day 3 — AI agents

- Card replacement workflow with execution-step tracking
- Mandatory human approval and idempotent action semantics

## Day 4 — Model Context Protocol

- JSON-RPC MCP endpoint supporting initialize, tools, resources, prompts, and ping
- Governed card-status and card-freeze capabilities

## Day 5 — Production RAG baseline

- Version-preserving document ingestion and metadata filtering
- Prompt-injection boundary, insufficient-context behavior, trace IDs, and tests
- Architecture, security boundaries, local runtime, and operational endpoints
